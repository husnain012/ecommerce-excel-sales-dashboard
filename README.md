# E-Commerce Sales & Returns Dashboard (Excel)

## Overview
This project analyzes an e-commerce transaction dataset using Microsoft Excel to evaluate sales performance, customer geography, product trends, and revenue loss due to returns.  
The focus is on practical business insights and real-world data handling.

---

## Tools & Techniques
- Microsoft Excel  
- Power Query (data cleaning & transformation)  
- PivotTables & Pivot Charts  
- Slicers for interactivity  

---

## Key Insights
- Calculated **Gross Sales, Return Loss, and Net Sales**
- Analyzed **monthly sales trends** using valid transaction dates
- Identified **Top 10 products by sales revenue**
- Identified **Top 10 countries by sales**
- Highlighted **Top 10 products with highest return revenue loss**
- Handled missing invoice dates using a **Date_Status approach** instead of deleting data
